## Landing Page – Rock Band (HTML & CSS Assessment)

## Description

THe shattered Riffs landing page designed for a rock band performance, The project highlights rock envoriments  through an elegant visual identity, utilizing advanced CSS techniques to provide a premium user experience. This site aims to capture the attention of  people who love listening rock music.

# Technologies Used

    HTML5: Semantic structure (header, main, section, article, nav, footer).
    CSS3: Advanced styling including:
    Flexbox and CSS Grid for layout management.
    Responsive Design (Media Queries).
    Custom Animations and Hover Effects.

## Project Structure

    index.html: The main semantic structure of the landing page.
    style.css: All layout rules, typography, and responsive adjustments.
    README.md: Project documentation in English.

## How to run the project

    Clone or download the repository.
    Navigate to the project folder.
    Open the index.html file in any modern web browser (Chrome, Firefox, Edge).
    No additional dependencies are required.

## Features Included

    grid-gallery: Detailed values using semantic unordered lists and display grid layout.
    Schedule Table: Complex data visualization using HTML tables (Extra Point).
    Multimedia Gallery: A responsive grid showcasing event highlights.
    Fully Responsive: Optimized for Mobile and Desktop (>1024px).
